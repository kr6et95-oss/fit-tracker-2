수면 + 러닝 자동 채우기

공식 API 없이 Connect 내보내기 파일로 휴식·운동을 채웁니다. 주관 점수(에너지·피로)는 그대로 두고, 측정값만 덮어씁니다.

  1. 러닝: Garmin Connect → 활동 선택 → 내보내기 → TCX 또는 GPX
  2. 수면: 날짜·시간 붙여넣기 또는 CSV (예: 2026-07-12 7.5 23:30)
  3. 미리보기 확인 후 적용 (같은 날짜 러닝은 병합·추가)
아직 불러온 항목이 없습니다.

FIT 원본은 Connect에서 TCX/GPX로 변환해 주세요. 수면 품질 점수가 CSV에 있으면 1–5로 환산합니다.

🔄 다른 기기와 맞추기

  1. 기기 A에서 백업 저장·공유 → iCloud/드라이브에 저장
  2. 기기 B에서 같은 파일을 연 뒤 복원 또는 병합
  3. 덮어쓰기는 현재 기기 데이터를 지우고, 병합은 날짜별로 합칩니다
