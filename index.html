수면 + 러닝 자동 채우기

공식 API 없이 Connect 내보내기 파일로 휴식·운동을 채웁니다. 주관 점수(에너지·피로)는 그대로 두고, 측정값만 덮어씁니다.

  1. 러닝: Garmin Connect → 활동 선택 → 내보내기 → TCX 또는 GPX
  2. 수면: 날짜·시간 붙여넣기 또는 CSV (예: 2026-07-12 7.5 23:30)
  3. 미리보기 확인 후 적용 (같은 날짜 러닝은 병합·추가)
