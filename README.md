# FIT TRACKER 2.2 · REST

> 체중 관리는 꾸준함이 답입니다.  
> **회복 > 영양 > 훈련**

1탄 감성을 이은 시퀄. **휴식 분석**을 최우선 지표로 둡니다.

## 온라인 (어디서나)

**https://kr6et95-oss.github.io/fit-tracker-2/**

아이폰 Safari로 접속 → 공유 → **홈 화면에 추가** 하면 앱처럼 쓸 수 있습니다.

> 식단·체중 등 데이터는 **그 기기 브라우저**에만 저장됩니다 (서버로 안 올라감).

## AI 코치 (Gemini)

공개 사이트에는 API 키가 포함되어 있지 않습니다.  
**더보기 → 설정**에서 [Google AI Studio](https://aistudio.google.com/apikey) 키를 넣으면 사진 분석·AI 코칭을 쓸 수 있습니다.

## 로컬 실행

```powershell
cd C:\Users\kr6et\fit-tracker-2
python -m http.server 5174
```

브라우저: **http://localhost:5174**

## 기능

| 영역 | 내용 |
|---|---|
| 😴 휴식 | 수면·에너지·근육 피로·스트레스 → **회복 점수 0–100** |
| 📊 리포트 | 주간/월간 KPI, 차트, 인사이트 |
| 🍽️ 음식 DB | 150+ 한국 음식, 탄단지 |
| 식단·체중·운동·물·습관 | 1탄 목표 공식 + 2탄 확장 |
| AI 코치 | 휴식 우선 코칭 (키 설정 시) |

## 1탄 데이터 가져오기

같은 브라우저에서 1탄을 썼다면:

1. 첫 화면 또는 **더보기 → 설정**
2. **「1탄 데이터 가져오기」**

`settings`, `goals`, `records`를 읽어 프로필·체중·식단을 이어받습니다. (사진은 용량 때문에 제외)
