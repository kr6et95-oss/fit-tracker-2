## 기능

| 영역 | 내용 |
|---|---|
| 😴 휴식 | 수면·에너지·근육 피로·스트레스 → **회복 점수 0–100** |
| 📊 리포트 | 주간/월간 KPI, 차트, 인사이트 |
| 🧬 체성분 | 체중 + **체지방% · 골격근 kg · 체수분%** 기록·추이 |
| 🏃 운동 | 유산소/근력 프리셋, **세트·reps·중량** 로그 |
| 👥 공유 | 주간 회복 카드 + **친구 공유 이미지·텍스트** |
| 💾 동기화 | 백업 덮어쓰기/**병합**, 기기 간 iCloud 동기화 가이드 |
| 📦 1탄 이전 | 자동 감지 미리보기, 병합 가져오기, JSON 파일 가져오기 |
| 🍽️ 음식 DB | 150+ 한국 음식, 탄단지 |
| AI 코치 | 휴식 우선 코칭 (키 설정 시) |

## 1탄 데이터 가져오기

같은 브라우저에서 1탄을 썼다면:

1. 첫 화면 배너 또는 **더보기 → 설정 → 1탄 → 2탄 마이그레이션**
2. 미리보기로 감지 내용 확인 후 **가져오기** 또는 **병합 가져오기**
3. 다른 기기였다면 **1탄 JSON 파일 가져오기**

`settings`, `goals`, `records`를 읽어 프로필·체중·식단을 이어받습니다. (사진은 용량 때문에 제외)

## 기기 간 맞추기 (계정 없이)

1. 기기 A: **더보기 → 설정 → 백업 저장·공유** → iCloud/드라이브 저장  
2. 기기 B: 같은 파일로 **병합 복원**(추천) 또는 덮어쓰기 복원
