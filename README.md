## 기능

| 영역 | 내용 |
|---|---|
| 😴 휴식 | 수면·에너지·근육 피로·스트레스 → **회복 점수 0–100** |
| 📊 리포트 | 주간/월간 KPI, 차트, 인사이트 |
| 🧬 체성분 | 체중 + 체지방% · 골격근 kg · 체수분% |
| 🍽️ 음식 DB | **400+** 한국 음식, **모든 항목 그램(g) 표기**, 탄단지 |
| 🏃 운동 | 유산소·근력·스포츠·회복 **100+ 종목**, 세트/reps/중량, 거리·RPE, 루틴 원탭 |
| 👥 공유 | 주간 회복 카드 + 친구 공유 이미지·텍스트 |
| 💾 동기화 | 백업 덮어쓰기/병합, 기기 간 iCloud 가이드 |
| 📦 1탄 이전 | 자동 감지 미리보기, 병합·JSON 파일 가져오기 |
| AI 코치 | 휴식 우선 코칭 (키 설정 시) |

## 음식 DB

- 카테고리: 주식, 단백질, 국찌개, 채소, 과일, 간식, 음료, 유제품, 편의점, 배달, 한식  
- 표기 예: `닭가슴살(구이) 100g (100g)` / `흰쌀밥 1공기 (210g)`  
- 분량 배수(0.5×~2×) 적용 시 **그램·칼로리·탄단지** 함께 스케일

## 운동 기록

- 모드: 유산소 / 근력 / 스포츠 / 회복  
- 근력 부위 필터: 가슴·등·어깨·하체·팔·코어·전신  
- 루틴 원탭: 푸시·풀·하체·전신·HIIT·회복  
- 추가 필드: 거리(km), RPE, 부위, 메모, 볼륨(kg)
