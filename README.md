## 가민 Forerunner 265

서버 계정 연동 없이 **Connect 내보내기**로 채웁니다.

### 러닝
1. 시계 활동이 Connect에 동기화된 뒤  
2. Connect → 해당 러닝 → **내보내기 → TCX 또는 GPX**  
3. 앱 **더보기 → 설정 → 가민 가져오기** 또는 **운동 탭**에서 파일 선택  
4. 미리보기 → 적용  

→ 날짜·시간(분)·거리(km)·kcal·페이스가 운동 기록으로 들어갑니다.

### 수면
**더보기 → 설정** 수면 붙여넣기 예:

```text
2026-07-12 7.5 23:30
2026-07-11 7h20m 00:10
7.5
```

- 날짜 없이 숫자만 쓰면 **오늘**로 저장  
- CSV에 수면 열·점수가 있으면 함께 해석 (점수는 1–5로 환산)  
- 이미 수면이 있으면 기본은 건너뛰고, **강제 덮어쓰기**로 교체 가능  

> FIT 원본 파일은 브라우저 제한으로 Connect에서 TCX/GPX로 받아 주세요.

## 음식 DB

- **약 1,000종+** (그램·단위·탄단지 포함)  
- 카테고리: 주식, 단백질, 국찌개, 채소, 과일, 간식, 음료, 유제품, 편의점, 배달, 한식, **카페, 베이커리, 분식, 반찬, 안주**  
- 표기 예: `흰쌀밥 1공기 (210g)` · `아메리카노(톨) 1잔 (300g)`  
- 분량 배수(0.5×~2×) 적용 시 **그램·칼로리·탄단지** 함께 스케일

## 운동 기록

- 모드: 유산소 / 근력 / 스포츠 / 회복  
- 근력 부위 필터: 가슴·등·어깨·하체·팔·코어·전신  
- 루틴 원탭: 푸시·풀·하체·전신·HIIT·회복  
- 추가 필드: 거리(km), RPE, 부위, 메모, 볼륨(kg)
